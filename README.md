# Consulting Slides Builder UI

Live demo: https://consulting-slides-builder.netlify.app/

This repository contains the frontend UI for **Consulting Slides Builder**. The UI provides a GPT-style experience where users paste pre-developed content, trigger an n8n webhook, and preview the generated slides as cards (Header, Title, Bullets, Suggested Graphs), with copy actions.

## Run Locally

### Prerequisites
- Node.js (recommended: LTS)
- npm (comes with Node)

### Install & Start
```bash
npm install
npm run dev
